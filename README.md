# Kenya Watch

Kenya Watch is a modern web-based platform for anti-corruption reporting and transparency in Kenya. It empowers citizens to anonymously report corruption, upload evidence (photos, videos, documents), track case status, and visualize trends. The platform integrates machine learning for text classification and pattern detection, supports multi-language and location-based reporting, and ensures user privacy and data protection.

## Features
- Anonymous corruption reporting
- Evidence upload (photos, videos, documents)
- Case tracking and status updates
- Machine learning for text classification and pattern detection
- Interactive maps and dashboards
- Multi-language and demographic support
- Whistleblower protection
- Integration with MongoDB
